Bismillah, saat tulisan ini dibuat, Alhamdulillah saya sudah berpindah kantor. Ini merupakan kali kedua saya berpindah kantor. Untuk ukuran developer abal-abal, Saya cukup setia membersamai kantor lama. Dua tahun setengah. Nah, kantor saya kali ini berada di Bandung. Itu membuat saya harus hijrah dari Jakarta ke Bandung. Onsite. Sesuatu yang tiba-tiba dan tidak pernah terpikirkan sebelumnya.

Singkat cerita, saya sebenernya sudah punya rencana. Setelah habis kontrak di Jakarta, saya akan mencoba mendalami dunia remote working (selain menikah juga tentunya 游때). Mengingat saya belum punya cukup banyak skill dan relasi, pilihan ini cukup sulit. Namun, belum ada keputusan untuk cari kantor baru. Pikiran saya waktu itu cuma ingin pulang kampung. Kangen ketemu keluarga dan si Dia. Rencana hanya akan jadi sebuah rencana. Kadang sama dengan kenyataan yang ada, kadang harus kita terima bahwa rencana itu bukan yang terbaik. Rencana saya tidak terealisasi.

Beberapa bulan sebelum waktu resign, saya berkenalan dengan seorang developer senior. Namanya Kang [Hendi Santika](https://github.com/hendisantika). Kita pertama kali komunikasi lewat facebook. Sebenernya saya sudah pernah kenal beliau secara tidak langsung lewat karya-karya beliau di Github. Kebetulan saya sering nongkrong di Github dang sering kepoin developer-developer hebat di sana. Nah dari situ, kita mulai berkomunikasi hingga akhirnya saya diajak join project keren di Kominfo Persis. Saya hobi berkontribusi. Project besar. Project kecil. Apapun itu selama ada kesempatan untuk berkontribusi. Mendapat tawaran join project dan menjadi relawan di sana, tentu membuat saya senang. Pake banget.

Saya banyak belajar. Walaupun project tersebut bersifat suka rela, Saya belajar bagaimana berkerja dengan banyak orang dan mendapatkan pengalaman yang luar biasa. Kemampuan git akan sangat dibutuhkan jika kita ingin bekerja dengan tim, tanpa tatap muka alias remote. Saya belajar bagaimana development project dengan [metode Agile](https://en.wikipedia.org/wiki/Agile_software_development) yang sesungguhnya. Di kantor lama saya belum pernah menerapkan development tersebut. Jadi harus beradaptasi. Namun, Saya suka dengan gaya development tersebut. Rapi dan terpantau. Saya jadi kenal beberapa orang yang semuanya sebagian besar orang Bandung. Circle saya menjadi lebih banyak. Awal mula saya akan pindah dari Jakarta ke Bandung.

Hampir tiap pekan selepas penat bekerja selama weekday, Saya sempakan buka gitlab. Project Kominfo Persis kebetulan ditempatkan di sana. Saya coba cek tiket-tiket yang masih open. Saya coba analisa. Apabila cocok, saya coba ambil dan assign. Mulai develop dan ketika selesai buat Merge Request. Cycle yang rapi. Lagi-lagi di kantor lama role model seperti ini tidak pernah saya dapatkan. Pengalaman yang luar biasa. Saya mulai enjoy dengan development seperti ini. Jelas dan terarah.

Nah, setelah beberapa pekan. Saya dihubungi kang Hendi. Mendadak dan sempat kaget juga. Saya diminta untuk join secara profesional dengan tim beliau. Tepat 1 bulan sebelum saya resign. Beliau butuh tim untuk handle project klien onsite. Saat itu saya mulai merenung. Tawaran tersebut tidak pernah terpikirkan sebelumnya. Kerja bareng senior developer secara profesional. Kalau kata orang sih seperti mimpi di siang bolong. Saya yang baru seumur jagung ini, Sempat tidak percaya. Tawaran tersebut akhirnya saya konsultasikan ke orang-orang terdekat.

Akhirnya, dengan pertimbangan yang cukup, Saya menerima tawaran tersebut. Itu artinya, saya harus siap melepas kantor saya saat ini dan juga teman-teman di sana. Dua tahun waktu yang cukup lama bersama mereka. Sempat mereka merasa sedih dan kehilangan atas keputusan saya ini. Tapi, hidup harus terus berlanjut dan saya telah mengambil keputusan 游때.

Petualangan baru pun dimulai. Semua barang perlengkapan dari Jakarta Saya kemas dan tentu akan berpindah ke Bandung. Semua proses seperti mencari kosan, beli perlengkapan untuk di Bandung akhirnya juga sudah dilakukan. Hidup baru part 2 游때. Bukan hanya itu, Petualangan baru karir saya pun baru dimulai. Saya akhirnya resmi sebagai seorang frontend developer setelah sebelumnya bertempur dengan menyandang status fullstack developer abal-abal 游때. Saya akan makin akrab dengan javascript, styling css, dan dom element. Kebetulan kantor baru mengadopsi Vue JS. Jadi harus semakin akrab juga dengan dia 游때. Bismillah...
